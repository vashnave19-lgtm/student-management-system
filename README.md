# 🚀 Student Management System

A console-based Student Management System built using Java and JDBC with Oracle Database.

## 📌 Features
- ➕ Add Student
- 📄 View All Students
- 🔍 Search Student by ID
- ✏️ Update Student Details
- ❌ Delete Student

## 🛠️ Tech Stack
- Java
- JDBC
- Oracle Database
- SQL

## ⚙️ How It Works
This application uses JDBC to connect Java with Oracle Database.  
All database operations are handled through a DAO (Data Access Object) layer.

## ▶️ How to Run

1. Compile:
- javac -cp ".;ojdbc8.jar" *.java
2. Run:
- java -cp ".;ojdbc8.jar" Main


## 📂 Project Structure
 
- Main.java → User Interface (Menu Driven)
- Student.java → Model Class
- StudentDAO.java → Database Operations


## 🎯 Key Concepts Used
- Object-Oriented Programming (OOP)
- JDBC Connectivity
- SQL (CRUD Operations)
- Exception Handling
